# Observability

## Metrics and SLOs
- API: latency p95 < 1s, error rate < 1%
- OCR: jobs/min, success/low-confidence/fail rates
- Flowable: process start success, task backlog

## Alerts
- 5xx spikes
- OCR failure rate threshold
- OCR backlog growth
- DB CPU
- Ingress 4xx/5xx

## Dashboards
- API latency/throughput
- OCR pipeline
- Workflow health
